# 🤖 Statbot-Pro

AI-powered CSV analysis tool using natural language queries.

## 🚀 Features
- Upload CSV files
- Ask questions in plain English
- AI-generated insights using Groq (Llama 3)
- FastAPI backend

## 🛠️ Tech Stack
- FastAPI
- LangChain
- Pandas
- Groq API

## ▶️ Run Locally

```bash
pip install -r requirements.txt
uvicorn main:app --reload
